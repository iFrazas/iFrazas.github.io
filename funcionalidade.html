Funcionalidades

O app seria direcionado somente para pessoas portadores da doença, e teria a funcionalidade de:

  • Estimulos cognitivos
  • Lembrete de tarefas diárias
  • Interface intuitiva / Acessibilidade auditiva (para aqueles que não sabem ler)
  • Sistema de login para cada usuário
  • Interação com o site, para que o cuidador possa planejar/mudar os lembretes de tarefas diárias
  • Sistema financeiro de doações
  • Local para publicação de artigos científicos ou resultados comprovados sobre pesquisas a respeito do Alzheimer (somente para médicos especializados, para que não seja publicado algo falso)
  • Banco de dados das informações obtidas através dos artigos científicos, para leitores
  • Sistema de login para cada usuário
  • Teste clínico online
  • Traduzido para vários idiomas
  • Tecnologias usadas
