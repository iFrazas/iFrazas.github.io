Funcionalidade do App

O app seria direcionado somente para pessoas portadores da doença, e teria a funcionalidade de:

- estimulos cognitivos

- lembrete de tarefas diarias

- interface intuitiva/ acessibilidade auditiva(para aqueles que não sabem ler)

- sistema de login para cada usuario

- interação com o site, para que o cuidador possa planejar/mudar os lembretes de tarefas diárias

- sistema financeiro de doações

- local para publicação de artigos científicos ou resultado, comprovador, sobre pesquisas a respeito do alzheimer( somente para medico especializado, para que não seja publicado algo falso)

- banco de dados, das informações optidas atraves do artigos científico, para leitores

- sistema de login para cada usuario

- teste clínico online

- traduzido para varios idiomas

- tecnologias usadas
